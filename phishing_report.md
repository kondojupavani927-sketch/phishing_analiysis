Task 2: Phishing Email Analysis Report

1. Email Overview
- Subject: IMMEDIATE ACTION REQUIRED: Security Alert on Your Account!
- Sender Name: Security Team
- Sender Email: support@secure-bank-alert-update.com

2. Red Flags Found
- Bad Greeting: It says "Dear Customer" instead of using a real name.
- Urgency: It threatens to close the account permanently in 24 hours.
- Fake Link: The link goes to a fake website domain called "login-update-verification-portal.net".

3. Header Analysis:
- SPF: FAIL - mailer.net not authorized by secure-bank-alert-update.com
- DKIM: None - No digital signature present
- Return-Path mismatch: Yes - Return-Path bounce@mailer.net ≠ From domain secure-bank-alert-update.com

4. Conclusion
This email is a phishing attempt. It uses scare tactics to force the user to click a suspicious link that mimics a bank login page.
